# CatHubAndroidClient

CatHubのAndroidクライアントアプリ、Kotlin LT 用に作成した。  
アプリを起動するとローカルのRailsサーバにあるAPIを叩き、画像URLを含むJSONを取得し、取得した画像URLから画像を表示。
現在は1枚までしか表示してない。


# 使ってるライブラリ

- [Retrofit](http://square.github.io/retrofit/)
- [Gson](https://github.com/google/gson)
